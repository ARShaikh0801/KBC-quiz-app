# KBC Quiz App

A quiz application inspired by "Who Wants to Be a Millionaire" / KBC format.

## Overview
Interactive quiz built primarily with JavaScript and styled with CSS. Presents questions, choices, and basic scoring/feedback.

## Features
- Multiple choice questions
- Score tracking
- Question navigation
- Simple lifelines or hints (if implemented)

## Tech stack
- JavaScript, HTML, CSS

## Running locally
1. Clone the repo:
   git clone https://github.com/ARShaikh0801/KBC-quiz-app.git
2. Open `index.html` in your browser, or serve locally:
   python -m http.server 8000
   open http://127.0.0.1:8000

## Customization
- Update the `questions` data file or JavaScript array to add/remove questions.
- Add persistence (localStorage) to save progress.
